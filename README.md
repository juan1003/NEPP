# NEPP
## Boilerplate for NEPP stack
The NEPP stack consists on the following tools:
- NextJS
- ExpressJS
- PrismaJS
- Postgres

You will need the following toolkit:
- Docker
- NPM
- NodeJS

To run this you will execute
```sh
cd NEPP
cd backend
npm install
cd ../frontend
npm install
cd ../
docker compose up -d
```

And that's it!

Happy Hacking!
